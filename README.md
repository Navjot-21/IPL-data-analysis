# IPL Data Analysis

##  Project Overview
This project performs Exploratory Data Analysis (EDA) on Indian Premier League (IPL) match data to uncover insights about teams, players, venues, toss decisions, and match outcomes. Various visualizations and statistical analyses are used to understand performance trends and identify key factors influencing matches.

---

##  Objectives

- Analyze IPL match data and player statistics.
- Identify top-performing teams and players.
- Study the effect of toss decisions on match outcomes.
- Explore venue-wise and season-wise trends.
- Visualize important patterns using charts and graphs.

---

##  Dataset

The dataset contains information such as:

- Match ID
- Season
- Teams
- Venue
- Toss Winner
- Toss Decision
- Match Winner
- Player of the Match
- Runs Scored
- Wickets Taken
- Batting and Bowling Statistics

##  Project Workflow

### 1. Data Collection
- Imported IPL datasets.

- Standardized column names.

### 3. Exploratory Data Analysis (EDA)

##  Key Insights

- Certain teams consistently perform better across seasons.
- Winning the toss can influence match outcomes under specific conditions.
- Some venues favor batting while others assist bowlers.
- Top batsmen and bowlers have a significant impact on match results.
- Player of the Match awards highlight the most influential performers.
